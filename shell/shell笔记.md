




##替换运算符

    $(varname:-word) 如果varname存在且非null，则返回其值；否则，返回word。
    用途：如果变量未定义，则返回默认值。
    范例：如果count未定义，则$(count:-0)的值为0

测试数值可以使用许多操作符，一般格式如下：

    [ "$number" -eq "100" ]
    
    -eq 数值相等。
    -ne 数值不相等。
    -gt 第一个数大于第二个数。
    -lt 第一个数小于第二个数。
    -le 第一个数小于等于第二个数。
    -ge 第一个数大于等于第二个数。
    下面的例子返回结果都一样。均为测试两个数是否相等（1 3 0是否等于1 3 0）。

    if [ "$ANS" = "y" ] || [ "$ANS" = "Y" ]
    then
      echo "   "
    else : #do nothing
    fi
