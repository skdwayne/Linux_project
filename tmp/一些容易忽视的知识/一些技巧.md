## 调试

    set -x  调试当前窗口命令
    set +x  取消调试

写脚本用全路径

# 软件

> 搜索某个软件是否安装

    rpm -qa |grep rsyslog
    rsyslog-5.8.10-10.el6_6.x86_64

> 搜索某个软件相关的文件

    rpm -ql  包名   查询软件相关文件
    rpm -qal

> 将光盘里的内容写到文件

    [root@localhost ~]# dd if=/dev/cdrom of=/root/CentOS-6.7-x86_64.iso

使用管道，如果写漏了不要紧  如果直接匹配的话就匹配不出来了
